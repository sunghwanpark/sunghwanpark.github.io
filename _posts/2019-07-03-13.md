Unity 표면셰이더
===============

## #pragma vert:(VertexFuntion)

### vertex funtion예제
<pre><code>
void vert (inout appdata_full v, out Input o)
{
  UNITY_INITIALIZE_OUTPUT(Input, o);
}
</code></pre>

### custom Input 예제
<pre><code>
struct Input
{
  float2 uv_texcoord;
  float fade;
};
</code></pre>

### Input의 기본 구조체 내용은 아래 링크에서 __서피스 쉐이더 인풋 구조__ 를 확인하면 된다
